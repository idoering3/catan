# Introduction

## Premise
Settlers of Catan is a popular board game. I wanted to make an expanded version with new features as an exercise in programming in Rust. What initially started as a basic project expanded in scope when I realized I wanted to make it have a user interface. 

This project quickly because an exercise in graphics programming. I hope to learn more using the wgpu library as well as document my process along the way through various commits. I hope to have a usable pseudo game engine by the end of the project, and hopefully I will be able to put the game on Steam for free. I plan on utilizing the vast majority of the code I write for a future video game project.
